## ฟังก์ชันหลัก

- Login 4 บทบาท
- Dashboard สรุปจำนวนเอกสาร มูลค่ารวม และสถานะ
- ใบขอซื้อ/ขอจ้างแบบการ์ด
- ค้นหาและกรองประเภท/สถานะ
- สร้างเอกสารใหม่พร้อมรายการพัสดุหลายรายการ
- Workflow สถานะงาน
- รายละเอียดเอกสาร
- พิมพ์/บันทึก PDF รายงานขอซื้อ/จ้าง และรายงานสรุป
- Export CSV
- ผู้ขาย/ผู้รับจ้าง
- Audit log
- ตั้งค่าหน่วยงาน
- เก็บข้อมูลใน LocalStorage

## หมายเหตุ

เวอร์ชันนี้เป็น Static Frontend Prototype ใช้งานบน GitHub Pages ได้ทันที ข้อมูลเก็บในเครื่องผู้ใช้ผ่าน LocalStorage ก่อน เหมาะสำหรับทดลอง Flow/UX ก่อนต่อฐานข้อมูลจริง เช่น Google Sheets, Firebase, Supabase หรือ Backend API
